
Uninstall files for ESRI Mapbook Developer Sample (DSMapbook).

	_UNINSTALL.bat
	DSMapBookPrj.dll
	DSMapBookUIPrj.dll
	de-register_component_category.reg

To use simply run _uninstall.bat
	
Should work for any version of mapbook from ArcMap 8.3 through 9.2 service pack 4, from the Arcmapbook project site or the ESRI Developer Network. If it doesn't please file a bug at http://code.google.com/p/arcmapbook/issues/list
